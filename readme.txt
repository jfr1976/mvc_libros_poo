Este proyecto es un CRUD en el cual se nos muestra en el index un listado de los libros.
Dentro de este listado tenemos las opciones de ver (para ver los detalles individuales del libro), editar y eliminar (por libro).
Tenemos una barra de navegación arriba donde se puede ver el listado de libros, el listado de editoriales y añadir un libro nuevo a la BD.
Se ha añadido en el controlador la verificación de datos para que no se pueda introducir codigo malicioso en nombre, descripción y número de páginas.

Se le ha dado el todo el proyecto un pequeño css para los estilos tanto del body como de la tabla y el formulario.

También se ha añadido un footer con el año y el nombre del autor.
